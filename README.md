# Order Subscription Service

Мини‑платформа для оформления заказов по подписке:

- Django REST API, Redis‑кеш и Celery‑воркер для уведомлений.
- Telegram‑бот, который связывает аккаунт по телефону.
- Docker Compose поднимает PostgreSQL, Redis, web и worker сервисы.
- Настроенный CI

Зависимости настроеные через Poetry, можно установить командой: `make deps`.
Запуск проекта: `make dev` - собирает образ в контейнере, поднимает БД и web-app, прогоняет линтеры, миграции и тесты.
Поднять - `make up` и остановить - `make down` контейнер.
Запуск сервера `make run` В Makefile прописаны все команды.
